Um chatbot simples, funcional e interativo desenvolvido em na linguagem python, utilizando Streamlit para a interface web e a API da OpenIa para gerar respostas inteligentes.

Este projeto foi desenvolvido por **Joana da Silva**, com foco em aprendizado de IA generativa, construção de interfaces simples e integrações entre APIs.

---
O código possui:

- Interface amigável com **Streamlit**
- Histórico de mensagens via st.session_state(devidamente configurado)
- Botão para limpar a conversa
- Integração com a API da OpenIa
- Conversa em formato de chat
- Uso do modelo gpt-4o-mini
---

Tecnologias presentes no código:

- Python 3.10+
- Streamlit  
- OpenAI API  
- dotenv
